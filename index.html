FabCon 2026 March 20, 2026

FabCon 2026: Microsoft Fabric's Agentic Era Begins

Over 8,000 data professionals gathered in Atlanta for the biggest FabCon yet — with major announcements around MCP, data agents, OneLake interoperability, and 31,000+ Fabric customers worldwide.

🤖
Fabric Data Agents — Now Generally Available

Fabric data agents are now GA, acting as virtual analysts that provide domain-specific insights. Combined with the new Fabric Local MCP (GA) and Fabric Remote MCP (preview), AI coding assistants like GitHub Copilot, Claude, and Cursor can now connect securely and directly to Fabric.

🔗
OneLake Interoperability Expands

OneLake now natively integrates with Azure Databricks via Unity Catalog (preview) and Snowflake (GA). New mirroring for SharePoint lists, Dremio, Oracle, and SAP Datasphere — plus Change Data Feed on top of mirrored data — makes OneLake the universal data hub.

📊
31,000+ Customers on Fabric

Microsoft Fabric has become one of the fastest-growing data platforms in Microsoft history, surpassing 31,000 customers. The ecosystem is accelerating with new ISV workloads, governance tools, and enterprise-grade AI capabilities across every workload.

Read the key announcements from FabCon 2026
Power Platform March 2026

Power Platform March 2026 Feature Update

Deeper Copilot experiences, vibe coding for Power Apps, unified modern controls, and Object-Centric Process Mining — March brings major leaps across the entire Power Platform stack.

🎨
Vibe Coding — Build Power Apps from a Prompt

A new vibe.powerapps.com preview enables developers to build full code Power Apps from a prompt using AI-driven plan, data, and app generation. Describe what you need, and the platform generates a complete, functional application.

🎛️
Unified Modern Controls — Quality Overhaul

All nine modern controls in canvas apps now share a unified property model with standardized names and typed enum values. Refined OnChange behavior reduces unnecessary recalculations, making apps faster and more responsive.

🔄
Object-Centric Process Mining

A new approach to process analysis where a single event can belong to multiple objects and types — orders, invoices, deliveries, payments — preserving the full web of interactions and dependencies end-to-end.

Read the full March 2026 update on Power Platform Blog
Microsoft Fabric March 2026

Fabric March 2026 Feature Summary

Major enhancements across governance, data engineering, real-time intelligence, and AI — Git selective branching, DLP policies for Fabric, and a new workload management hub in the Admin Portal.

🔀
Git Integration — Selective Branch-Out

Developers can now select only the items they need when branching out in Fabric Git Integration. Work with a smaller, purpose-built workspace to iterate faster, reduce clutter, and improve reliability across your CI/CD pipeline.

🛡️
Data Loss Prevention Policies for Fabric

DLP policies now extend to Fabric in preview, giving admins the ability to define and enforce data protection rules across the platform. Combined with the new Manage Workloads tab in the Admin Portal, governance has never been more centralized.

⚡
Real-Time Stream Processing & Snowflake Mirroring

March brings real-time stream processing capabilities in preview alongside extended mirroring with change delta feed and Snowflake support — enabling live, always-current analytics across hybrid data estates.

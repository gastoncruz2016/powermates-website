Power Mates at FabCon Atlanta

Session 1 ✓

Building Next-Gen Apps with Power Platform and SQL Server 2025

Demonstrated how Power Apps, Dataverse, and Fabric work together for real-time enterprise applications. Covered Code Apps (now GA), Copilot in model-driven apps, and the Power Apps MCP Server.

Key Announcements

🤖

AI Agents Go Native

Fabric Data Agents hit GA. Fabric Remote MCP lets AI agents perform authenticated actions in Fabric. The agent era for data platforms is here.

🔗

OneLake Becomes the Hub

Databricks Unity Catalog reads OneLake natively (preview). Snowflake interop is GA. OneLake security with row + column level controls goes GA in weeks.

📊

35M+ Power BI Users

Microsoft confirmed 35+ million active Power BI users and 31,000+ Fabric customers. Fabric is the fastest-growing data platform in Microsoft's history.

⚡

Runtime 2.0 & Spark 4

Next-gen compute: Apache Spark 4.x, Delta Lake 4.x, custom live pools, resource profiles, and Z-Order support — all GA. Materialized lake views also GA.

🏗️

Git & DevOps Overhaul

Branched workspaces, selective branching, code comparison, and Fabric CLI v1.5 for CI/CD. Deploy a workspace in a single terminal command.

🔄

Mirroring Everywhere

Oracle and SAP mirroring hit GA. SharePoint lists, MySQL, and Dremio in preview. 1,000+ table support. Change Data Feed and Snowflake view support added.

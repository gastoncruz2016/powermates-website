FAQ

Common Questions

How is this different from the $7,500 Enterprise tier?
Enterprise uses Power BI Admin APIs for a comprehensive but static analysis. AI-Assisted connects directly to Fabric REST APIs using Microsoft's FabricAdmin agent skills, scanning item types that Admin APIs don't cover (Lakehouses, Warehouses, Notebooks, Pipelines, KQL databases, Eventstreams). Plus you get a live 2-hour working session instead of a report walkthrough.
Do you need admin access to my tenant?
We need a service principal with read-only Fabric API permissions. We provide a step-by-step setup guide. The principal has zero write access.
How long does the scan take?
The automated scan runs 15–45 minutes depending on tenant size. The full engagement (discovery + scan + live session + report delivery) is typically completed within one week.

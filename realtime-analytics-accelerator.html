Architecture

The Streaming Pipeline

From raw events to live dashboards — all within Fabric, no external infrastructure.

DATA SOURCES       →  EVENTSTREAM          →  KQL DATABASE          →  DASHBOARD
  Event Hub              Ingest + route           RawEvents                 Real-Time Dashboard
  IoT Hub                Filter + transform       ParsedEvents (update)     Activator Alerts
  Custom App / REST      Fan-out to targets       Materialized Views        Teams / Email
  Kafka                  Dead-letter queue        Retention + caching       Power Automate
